# Agentic Workflow Automation

A full-stack application designed to build, manage, and automate AI-driven agentic workflows. This project features a robust architecture split between a user-facing frontend and an intelligent backend, streamlined for deployment on Hugging Face.

## 🚀 Project Structure

The repository is organized into two main environments to separate concerns and simplify development:

* **`/frontend`**: Contains the client-side application. Built primarily with TypeScript, alongside CSS and standard web technologies, it provides the interactive user interface for defining and monitoring workflows.
* **`/backend`**: Contains the core automation logic and APIs. Built with Python, this directory houses the agentic processes, language model integrations, and routing necessary to execute complex tasks.

## 🛠️ Tech Stack

* **Frontend Ecosystem**: TypeScript, JavaScript, HTML, CSS
* **Backend Ecosystem**: Python
